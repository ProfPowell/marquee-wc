This <marquee-wc> component is a marquee tag reborn for the year 2026. 

It is accessible
It uses CSS where possible a standard <marquee> could but uses CSS vars for styling
It has many other modes and styles for your modern and retro pleasure
It emits custom events to subscribe to and as a component can be scripted and style like anything else on the web platform

From a scrolling ticker, dot matrix read out, or annoying web effect the <marquee-wc> can do it all!
